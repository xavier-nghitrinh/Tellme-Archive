TellMe Export
=============

HOW TO VIEW:
  1. Extract ALL files from this ZIP, keeping the folder structure intact.
  2. Open index.html in a browser (Chrome, Safari, Firefox).

On Android : use ZArchiver or similar to extract, then open index.html with Chrome.
On iOS     : use the Files app to extract, then open index.html with Safari.
On PC      : extract and double-click index.html.

To publish online: upload all extracted files to any static host (e.g. GitHub Pages).
Images will display on all devices once hosted.

FILE STRUCTURE:
  index.html          — viewer UI
  meta.js             — all captions, transcripts, dates
  images/YYYY/MM/DD/  — original photos
